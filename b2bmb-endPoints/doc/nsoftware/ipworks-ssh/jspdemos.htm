IP*Works! SSH V9 Java Edition  -  Server Demos

NOTE: These pages are simple demos, and by no means complete applications. They are intended to illustrate the usage of the IP*Works! SSH components in a simple, straightforward way. What we are hoping to demonstrate is how simple it is to program with our tools. If you want to know more about them, or if you have questions, please visit www.nsoftware.com or email to support@nsoftware.com.

The following is the list of demos with short descriptions:

psclientgetprocess.jsp This demo uses the PSClient component to run the get-process command on a remote machine that is running the PowerShell Server.
psclientgetservice.jsp Shows how to use the PSClient component to list, start, and stop services on a host running PowerShell Server.
psclientsearchfiles.jsp This demo shows how to the use the PSClient component to search files in a specified path for a given search term on a remote machine that is running the PowerShell Server.
sftp.jsp A full featured SFTP client built using the SFTP component. It allows browsing of directories, uploads and downloads of files, and more.
