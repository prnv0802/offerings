IP*Works! SSH V9 Java Edition  -  Desktop Demos

NOTE: These projects are mostly simple demos, and by no means complete applications. They are intended to illustrate the usage of the IP*Works! SSH components in a simple, straightforward way. What we are hoping to demonstrate is how simple it is to program with our tools. If you want to know more about them, or if you have questions, please visit www.nsoftware.com or email to support@nsoftware.com.

Java GUI Demos

Certificate Manager Full-featured certificate management sample. Demonstrates how to list, manage, create, and sign certificates.
Remote Execution (SExec) Shows how to use the SExec component to remotely and securely execute commands.
Secure Shell (SShell) Shows how to use the SShell component to remotely and securely execute commands.
SFTP Client A full featured SFTP client built using the SFTP component. It allows browsing of directories, uploads and downloads of files, and more.
SSHClient This demo shows how to use the SSHClient to establish a secure connection to any SSH host.
SSHTunnel Uses the SSHTunnel component to accept incoming plain text connections and tunnel them through SSH.

Java Console Demos

PSClient Get-Process Demo This demo uses the PSClient component to run the get-process command on a remote machine that is running the PowerShell Server.
PSClient Get-Service Demo Shows how to use the PSClient component to list, start, and stop services on a host running PowerShell Server.
PSClient Search Files Demo This demo shows how to the use the PSClient component to search files in a specified path for a given search term on a remote machine that is running the PowerShell Server.
Remote Execution (SExec) Shows how to use the SExec component to remotely and securely execute commands.
SCP Client A full featured SCP client built using the SCP component. It allows browsing of directories, uploads and downloads of files, and more.
SFTP Client A full featured SFTP client built using the SFTP component. It allows browsing of directories, uploads and downloads of files, and more.
SFTP Server Demo Shows how to build a simple SFTP Server.
