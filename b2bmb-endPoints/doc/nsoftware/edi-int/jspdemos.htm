EDI Integrator V9 Java Edition  -  Server Demos

NOTE: These pages are simple demos, and by no means complete applications. They are intended to illustrate the usage of the EDI Integrator components in a simple, straightforward way. What we are hoping to demonstrate is how simple it is to program with our tools. If you want to know more about them, or if you have questions, please visit www.nsoftware.com or email to support@nsoftware.com.

The following is the list of demos with short descriptions:

as2server.jsp A simple example of an AS2 server.
asyncmdn.jsp Sample code illustrating how to receive an asynchronous MDN.
gisbserver.jsp Shows how to use the GISBReceiver component.
rnifserver.jsp A simple example of a RosettaNet server.
