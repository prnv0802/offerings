EDI Integrator V9 Java Edition  -  Desktop Demos

NOTE: These projects are mostly simple demos, and by no means complete applications. They are intended to illustrate the usage of the EDI Integrator components in a simple, straightforward way. What we are hoping to demonstrate is how simple it is to program with our tools. If you want to know more about them, or if you have questions, please visit www.nsoftware.com or email to support@nsoftware.com.

Java GUI Demos

Certificate Manager Full-featured certificate management sample. Demonstrates how to list, manage, create, and sign certificates.
OFTP Server An OFTP server showing how to receive and send files.
RosettaNet Client A simple but full-featured RosettaNet client.
Secure FTP Client A secure FTP client built using the FTPS component. It allows browsing of directories, uploads and downloads of files, and more.

Java Console Demos

AS1 Sender/Receiver Demo Shows how to use the AS1Sender component to send an EDI document, and the AS1Receiver to process it.
AS2 Client A simple but full-featured AS2 client.
Certificate Manager Full-featured certificate management sample. Demonstrates how to list, manage, create, and sign certificates.
EDI Parser Shows how to parse EDI data via the ParseFile or Input methods as well as through events.
GISB Client A simple client for sending EDI data using the GISB protocol.
OFTP Client A simple OFTP client which uploads and downloads files to and from the server.
Secure FTP Client A secure FTP client built using the FTPS component. It allows browsing of directories, uploads and downloads of files, and more.
